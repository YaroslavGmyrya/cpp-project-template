# Шаблон для C++ проекта

Шаблон влючает:

1. Автоматическую генерацию документации через Doxygen
2. Автоматическую коррекцию стиля кода после коммита
3. Автоматическую сборку с помощью cmake
4. Скрипты для автоматической установки зависимостей, сборки/пересборки проекта
5. Автоматическую установку Google test и spdlog с помощью cmake 
6. Стандартную структуру C++ проекта (директории include, src, tests)
7. Github Action сборку и тест при пуше с тегом. Используется github self-hosted runner на своем сервере
